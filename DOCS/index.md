# Welcome to SAP FCM Training Sandbox

## Project Overview
This project is a **Risk-Free SAP FI/CO Practice Simulator** designed to help users learn SAP Financials and Controlling without the fear of breaking a live system.

## Documentation
- [Product Requirements Document (PRD)](PRD.md): Detailed project requirements and features.
- [Tutorial](Tutorial.md): Step-by-step guide to using the sandbox.
- [Tasks](Tasks.md): List of tasks and to-dos for the project.
- [Presentation](presentation/index.html): Project presentation slides.

## Key Features
- **Zero Risk**: Isolated sandbox environment.
- **Visualization**: End-to-End process flow visualization.
- **Auto Feedback**: Immediate error guides and explanations.
